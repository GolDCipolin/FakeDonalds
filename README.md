# Mobile Computing Assessment 24/25

FakeDonalds App

This is an app where users can go and buy food from a restaurant called FakeDonalds.
The app allows users to view the menu, add items to their cart, and place an order.
The app also allows users to view their order history and the status of their current order.
Users can choose from a variety of order placements, such as delivery, pickup, and dine-in.

## Features
- Camera: Scanning QR codes for table dine.
- Microphone: Speech to text for the address in checkout page.
- Location: Orders based on the closest shop nearby.
- Vibration: Vibrates when user adds food or drink to the basket.
- Text-to-speech: Can be turned on from the account page and will read out anything the user does.

## Future Improvements
- Add a payment systems that will accept credit/debit cards, apple pay, paypal.
- Loyalty points system.
- Add a search bar to the food page.
- Being able to change email/password/username for own account.
